HTML이란 무엇인가?

Hypertext Markup Language(HTML)는 하이퍼 텍스트와 콘텐츠를 표시하는 언어. 콘텐츠를 표시하는 방식으로 웹페이지를 만든다. 즉, html은 웹브라우저를 통해 표시되는 웹페이지의 콘텐츠를 정의하기 위해 사용하는 언어.

HTML 은 언어이다 Html 언어를 목적에 맞게 사용하기 위해서는 HTML의 문법에 맞는 표현을 사용해야 한다 HTML : 태그(tag) 만 기억하자! * 태그란? HTML 코드에서 정보(콘텐츠)를 정의하는 형식


HTML 태그 사용법 : 시작과 끝
태그는 <> , 기호를 사용해 콘텐츠의 시작과 끝을 표시 각 태그는 콘텐츠를 감싸며, 태그명은 콘텐츠의 성격과 의미를 나타냄 태그사용형태 : <태그명>여기에 콘텐츠 기입
